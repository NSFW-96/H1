## AI Integration Setup

To enable AI-powered health analysis, you need to set up a Nebius AI API key:

1. Copy `.env.example` to `.env.local` in the project root directory
2. Get your API key from Nebius AI Studio (https://studio.nebius.com/)
3. Add your Nebius API key to the `.env.local` file:
   ```
   NEBIUS_API_KEY=your-api-key-here
   ```

The health assessment will now use the Phi-4 model for AI-powered analysis.
